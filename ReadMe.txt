tutorial git-hub mantap:
1. buat Repository baru di http://github.com
2. clone (pojok kanan) copy URL
3. pada Git Dush ketik
a. Masuk directory yang ingin dibuat Clone
b. git clone URL
c. Masuk directory yang telah dibuat
d. Buah sebuah file
e. git add .
f. git commit -m "publish file namafile"
g. git push origin master
